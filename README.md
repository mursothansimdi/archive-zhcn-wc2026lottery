# archive-zhcn-wc2026lottery

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态 HTML 文件的形式存放，不针对任何单个域名或具体网站。

## 项目简介

archive-zhcn-wc2026lottery 是一个通用型 HTML 页面归档仓库。仓库内保存的 HTML 文件均为独立页面，可直接在浏览器中打开查看。仓库本身不提供动态服务，也不依赖特定后端环境。

## 目录说明

- `index.html` — 仓库入口页面，提供对归档内容的概览导航。
- `pages/` — 存放主要 HTML 页面文件的目录。
- `assets/` — 存放页面引用的静态资源文件（如 CSS、JavaScript、图片等）。
- `archive/` — 存放历史版本或备用的 HTML 页面。

目录结构可能根据归档内容的变化进行调整，请以实际仓库文件为准。
